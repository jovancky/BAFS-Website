# **App Name**: LedgerLeague

## Core Features:

- Interactive Journal Entry: Simulate journal entries with real-time feedback on debit and credit inputs.
- Gamified Challenges: Offer time-bound challenges and daily streaks for consistent practice.
- Real-world Scenarios: Present business scenarios where user decisions impact journal entries and financial statements.
- AI-Driven Personalized Tutoring: Uses LLM tool to create a learning path adapting to the user's performance and comprehension levels.
- Progress Dashboard: Visually track progress with performance analytics, highlighting areas for improvement.
- Quizzes and Flashcards: Multiple-choice quizzes and flashcards to test knowledge of key concepts.
- Avatar Customization: Allow users to personalize their learning experience via customizable avatars and profiles.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5), evokes trust and stability.
- Background color: Light Grey (#F5F5F5), provides a clean, non-distracting backdrop.
- Accent color: Teal (#009688), complements the primary color while standing out for interactive elements.
- Body: 'Inter', a sans-serif font providing a modern, readable feel suitable for long blocks of text.
- Headlines: 'Space Grotesk', a sans-serif font bringing a slightly futuristic, scientific touch; should only be used where blocks of text are brief.  When 'Space Grotesk' is in use for the headline, 'Inter' should be used for the body.
- Use financial and accounting-themed icons, such as charts, graphs, and currency symbols.
- Maintain a clean, intuitive layout that emphasizes learning through clear sectioning and minimal clutter.
- Use subtle animations to visualize accounting equations and illustrate financial processes, enhancing comprehension.
